# Flows Garden

Flows Garden is a premium editorial landing page for a botanical brand, built to feel like a cultivated journal rather than a generic marketing site. The visual language balances structured layout with organic irregularity through asymmetric composition, soft motion, and botanical imagery.

## Design Decisions

- Layout: the page uses full-width editorial sections with uneven grids, overlap, and bleed to keep the composition intentional but not rigid.
- Color: the surface stays pale lime cream, with deep forest green for primary text, warm earth brown for supporting tones, and dark red for the conversion section.
- Typography: Newsreader is used for headlines to give the page an elegant, literary voice, while Plus Jakarta Sans keeps body copy clean and readable.
- Motion: Framer Motion adds subtle fade-and-rise entrance effects so the page feels alive without becoming distracting.
- Image treatment: botanical visuals use asymmetric leaf masks and ghost borders instead of hard rectangular framing.

## Tech Stack

- Next.js App Router: provides the page structure, routing, and image optimization for a production-ready landing page.
- Tailwind CSS: keeps the layout responsive and compact while still allowing custom editorial styling.
- Framer Motion: handles the staggered section and card animations.
- next/font/google: loads Newsreader and Plus Jakarta Sans with optimized font delivery.
- next/image: keeps the botanical visuals responsive and performant.
